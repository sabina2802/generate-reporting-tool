# ReportCraft

A versatile reporting tool that transforms raw data into professional, interactive reports with customizable templates, automated data visualization, and multi-format export capabilities for business intelligence and decision-making.

## Tech Stack
- **Frontend**: Next.js 14, TypeScript, Tailwind CSS
- **Backend**: Laravel 11, PHP 8.2, Sanctum Auth
- **Database**: MySQL 8
- **Admin**: AdminLTE 3

## Modules
- 🔐 **Authentication**: User login, registration, role-based access control, and session management for secure access to reports
- 📊 **Report Builder**: Drag-and-drop interface to create custom reports with charts, tables, filters, and formatting options
- 🗄️ **Data Sources**: Connect to databases, APIs, CSV files, and other data sources with real-time synchronization
- 📈 **Dashboard**: Interactive dashboard displaying key metrics, KPIs, and visual summaries of important data
- 📤 **Export & Sharing**: Export reports to PDF, Excel, CSV formats and share via email, links, or scheduled delivery
- ⏰ **Report Scheduling**: Automated report generation and distribution on daily, weekly, monthly, or custom schedules

## Quick Start
```bash
# Clone
git clone https://github.com/YOUR_USERNAME/generate-reporting-tool.git
cd generate-reporting-tool

# Backend setup
cd backend-laravel
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
# → runs on http://localhost:8000

# Frontend setup (new terminal)
cd frontend-nextjs
npm install
npm run dev
# → runs on http://localhost:3000
```

## API Endpoints
- `GET /api/users` — Get all users
- `POST /api/users` — Create new user
- `GET /api/data-sources` — Get all data sources
- `POST /api/data-sources` — Create new data source
- `GET /api/reports` — Get all reports
- `POST /api/reports` — Create new report
- `GET /api/reports/:id/run` — Execute report and return data
- `GET /api/dashboards` — Get all dashboards
- `POST /api/dashboards` — Create new dashboard
- `GET /api/scheduled-reports` — Get scheduled reports
- `POST /api/scheduled-reports` — Create scheduled report
- `POST /api/reports/:id/export` — Export report to PDF/Excel
- `GET /api/exports` — Get export history
- `PUT /api/reports/:id` — Update report configuration
- `DELETE /api/reports/:id` — Delete report
